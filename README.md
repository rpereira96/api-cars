
API de aluguel de carros usando TypeScript, seguindo os princípios de S.O.L.I.D., testes de unidade e testes de integração.

TECNOLOGIAS
 * NodeJs

 * Express

 * Typescript

 * Docker

 * PostgreSQL

 * Swagger

 * SOLID 

INSTALL
  - yarn

Usage
# CRIAR E STARTAR CONTAINERS
  - docker-compose up

# EXECUTAR migrations
  - yarn typeorm:migration-run

DOCUMENTAÇÃO
   - http://localhost:[PORT]/api-docs/
